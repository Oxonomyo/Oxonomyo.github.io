Les maisons aux âmes perdues

Guide d'instalation et de désinstalation

Windows

1.1 Introduction
Ce manuel vous guidera pas à pas dans l’installation du jeu sur votre ordinateur.
1.2 Étape 1 : Téléchargement du jeu

— Rendez-vous sur https ://unununium-company.github.io/UnununiumWeb/telechargement.html

et téléchargez le fichier d’installation ou bien, ouvrez la clé USB fournie.

— Assurez-vous de choisir la version compatible avec votre système d’exploitation (Windows, macOS ou Linux).

1.3 Étape 2 : Lancement du fichier d’installation

— Double-cliquez sur le fichier d’installation téléchargé.

— L’assistant d’installation va se lancer. Suivez les instructions à l’écran.

1.4 Étape 3 : Choix du type d’installation

— Sélectionnez si vous souhaitez installer le jeu pour tous les utilisateurs de l’ordinateur ou uniquement pour votre compte.

— Cliquez sur "Suivant".

1.5 Étape 4 : Choix de la langue de l’aide

— Sélectionnez la langue que vous souhaitez utiliser pour l’aide du jeu.

— Cliquez sur "Suivant".

1.6 Étape 5 : Choix du répertoire d’installation

— Par défaut, le jeu sera installé dans le répertoire

"C :/Program Files/ les_maisons_aux_ames_perdues ". Vous pouvez modifier ce répertoire si vous le souhaitez.

— Cliquez sur "Suivant".

1.7 Étape 6 : Options d’installation

— Cochez la case "Créer une icône sur le bureau" si vous souhaitez avoir un raccourci vers le jeu
sur votre bureau.

— Cliquez sur "Installer".

1.8 Étape 7 : Fin de l’installation

— L’installation du jeu va démarrer.

— Une fois l’installation terminée, un message de confirmation s’affichera.
— Cliquez sur "Fermer".

1.9 Étape 8 : Lancement du jeu

— Vous pouvez maintenant lancer le jeu en double-cliquant sur l’icône sur le bureau ou en le recherchant dans le menu Démarrer.

Félicitations ! Vous avez terminé l’installation du jeu. Vous pouvez maintenant profiter de toutes ses fonctionnalités.

2.1 Introduction

Ce manuel vous guide pas à pas pour désinstaller le jeu de votre ordinateur.

2.2 Prérequis

— Assurez-vous d’avoir les droits d’administrateur sur l’ordinateur.

— Fermez toutes les instances du jeu avant de commencer la désinstallation.

2.3 Instructions

— Ouvrez le menu Démarrer de Windows.

— Dans la barre de recherche, tapez le nom du jeu.

— Sélectionnez le jeu dans les résultats de la recherche.

— Cliquez sur le bouton "Désinstaller" qui apparait à côté du nom du jeu.

— Windows vous demandera de confirmer la désinstallation. Cliquez sur "Oui" pour continuer.

— Le processus de désinstallation démarrera. Suivez les instructions à l’écran pour le terminer.

— Une fois la désinstallation terminée, un message de confirmation s’affichera. Cliquez sur "Fermer" pour quitter l’assistant de désinstallation.

2.4 Vérification de la désinstallation

Pour vous assurer que le jeu a été désinstallé correctement, vous pouvez :

— Vérifier que le dossier du jeu a été supprimé de votre ordinateur. Le dossier du jeu se trouve généralement dans le répertoire C :/Program Files ou C :/Program Files (x86).

— Rechercher le jeu dans la liste des programmes installés dans le Panneau de configuration de Windows. Si le jeu n’apparaît pas dans la liste, il a été désinstallé avec succès.

Vous avez désinstallé avec succès le jeu de votre ordinateur.
